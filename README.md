## Опис

**DevCourses** — платформа IT-курсів для студентів і розробників.
Сайт демонструє адаптивний інтерфейс з картками курсів, бургер-меню та CSS-анімаціями.

## Структура проєкту

```
lab1/
├── index.html       # Головна сторінка
├── css/
│   └── style.css    # Стилі (flexbox, grid, media queries, анімації)
├── js/
│   └── main.js      # Бургер-меню, scroll-анімації, активна навігація
└── assets/          # Графічні ресурси
```

## Запуск

Відкрийте `index.html` у браузері — жодних залежностей не потрібно.

## Реалізовано

- Семантична HTML-розмітка (`header`, `main`, `section`, `footer`, `article`)
- Адаптивна сітка курсів: 3 колонки → 2 → 1 (grid + media queries)
- Бургер-меню для мобільних пристроїв (CSS + JS)
- Scroll fade-in анімації (IntersectionObserver)
- Підсвічування активного пункту меню при скролі
- Hover-ефекти на картках та кнопках
- Відносні одиниці: `rem`, `em`, `vw`, `clamp()`
- Media queries: `≤1024px`, `≤768px`, `≤480px`
